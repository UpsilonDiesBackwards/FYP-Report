    {\bfseries\Large Abstract}\\[0.5em]
    \begin{spacing}{1.2}

   This report investigates the accessibility of scientific literature from a software engineering perspective, using the creation of a sub-atomic particle physics simulation
   as a case study. Using physics research, a simulation will be made that aims to realistically reproduce the behaviour of the fundamental particles, including Protons,
   Neutrons, Electrons. Furthermore, the simulation will attempt to represent the four fundamental interactions: the strong, weak, electromagnetic, and gravitational forces.

   Through the implementation of these concepts without prior academic or professional expertise, this project aims to evaluate the translatability and level of understanding
   needed to take the mathematics and concepts discussed into code.
    \end{spacing}
